# 🏨 Hotel Booking Data Analysis

## 📌 Project Overview

This project analyzes hotel booking data to uncover patterns, trends, and insights related to customer behavior, booking cancellations, pricing, and seasonal demand.

The dataset used: **Hotel_Bookings.csv**

The analysis is performed using Python in Jupyter Notebook (`hotel_booking.ipynb`).

---

## 🎯 Objectives

* Analyze booking patterns and customer behavior
* Identify factors affecting booking cancellations
* Explore pricing trends
* Visualize relationships between reviews, price, and booking frequency
* Generate meaningful business insights

## 📊 Exploratory Data Analysis (EDA)

The following analysis was performed:

* Data Cleaning (Handling null & infinite values)
* Univariate Analysis
* Bivariate Analysis
* Price vs Room Type
* Reviews vs Price
* Booking Cancellation Analysis
* Distribution plots & Boxplots
* Correlation Heatmap

---

## 📈 Key Insights

* Higher-priced rooms tend to have fewer reviews.
* Certain room types dominate bookings.
* Booking cancellations depend on multiple factors like price, lead time, and customer type.
* Seasonal trends affect booking frequency.
